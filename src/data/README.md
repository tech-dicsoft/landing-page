# Cách chỉnh nội dung Highlighted Stories (Portfolio)

Chỉ cần mở file **`projects.json`** và sửa nội dung. Không cần biết lập trình.

- **title**: Tên dự án
- **description**: Mô tả ngắn về dự án
- **stack**: Danh sách công nghệ (có thể để `[]` nếu không có)
- **location**: Quốc gia/khu vực (có thể để `null` nếu không có)

Thêm dự án mới: copy một khối `{ ... }` trong mảng, dán xuống và sửa nội dung. Nhớ thêm dấu phẩy `,` giữa các khối.
